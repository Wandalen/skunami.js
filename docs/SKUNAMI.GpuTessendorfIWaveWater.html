GPU height field water simulation based on "Interactive Water Surfaces" (Jerry Tessendorf, Game Programming Gems 4)
Parameters:
Name Type Description
options object Options
Properties
Name Type Argument Default Description
renderer THREE.WebGLRenderer Three.js WebGL renderer
scene THREE.Scene Three.js scene
mesh THREE.Mesh Three.js mesh for sculpting
size number Size of mesh
res number Resolution of mesh
dampingFactor number Damping factor for the sim
multiSteps number <optional>
 1 Number of full steps to take per frame, to speed up some of algorithms that are slow to propagate at high mesh resolutions
meanHeight number <optional>
 0 Mean height of the water simulation

 1000 Density of water (kg per cubic metres)
